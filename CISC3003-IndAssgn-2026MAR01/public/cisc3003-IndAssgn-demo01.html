Pet Intelligent Management System

Period: May 2025 – Present   |   Role: Team Member   |   Technologies: Python, Elasticsearch, Spark

Project Overview

This ongoing project at the University of Macau aims to provide intelligent pet management services using big data and deep search technologies. The system helps pet owners track health, activities, and find lost pets via image recognition.

My Contributions

  • Participated in requirements analysis and designed the database schema.
  • Used Python (Pandas) for data cleaning and preprocessing of pet datasets.
  • Implemented fast pet information retrieval based on Elasticsearch.
  • Explored Spark Streaming for real-time activity monitoring.

Technologies Used

Python Elasticsearch Apache Spark Kibana MySQL Flask (planned)

Outcomes

The project is still in development, but we have successfully built a prototype that can index pet information and support basic search. Future work includes integrating image recognition for lost pet detection.
